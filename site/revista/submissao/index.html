Submissão de artigo

Canal para autores enviarem metadados, texto em Markdown e figuras. A redação aplica o template editorial e gera o PDF do artigo.

Envio pelo formulário (em fase de testes). A equipe editorial está validando o fluxo de submissão. Por enquanto, entre em contato com o editor para enviar seu manuscrito.

Usado no nome da pasta. Se vazio, é derivado do título.

Um autor por linha. Use tradução:, revisão: etc. para papéis.

Seções: ##. Figuras: ![legenda](figuras/nome.png). Tabelas: ![legenda](tabelas/nome.csv). Gráficos: ![legenda](graficos/nome.png) (envie também o CSV dos dados no zip de gráficos). Equações: LaTeX no texto, ex. $E = mc^2$ ou $$ ... $$.

Imagens na raiz do zip → pasta figuras/ (PNG, JPG, etc.).

Um arquivo .csv por tabela, na raiz do zip → pasta tabelas/.

Para cada gráfico: imagem (.png / .jpg) para o PDF e, se possível, .csv com os dados (referência para a redação; o layout final é ajustado manualmente).
