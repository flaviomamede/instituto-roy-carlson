# Versões guardadas

Esta pasta guarda **snapshots** dos documentos estratégicos e de planejamento, por data e origem, para que nenhuma versão se perca.

## Estrutura

- **`_versoes/AAAA-MM-DD/`** — Uma pasta por dia em que foi feita uma entrega ou revisão.
- Dentro: cópias dos arquivos como estavam naquele momento (ex.: `PESQUISA.md`, `RESUMO_EXECUTIVO.md`, `PLANO_ESTRATEGICO_INSTITUTO_ROY_CARLSON.md`).
- Em cada pasta de data pode haver um **README.md** opcional indicando qual agente/sessão gerou ou revisou (ex.: "Agente Planejamento Estratégico").

## Regra

- **Não apague** pastas antigas. Todas as versões têm algo a aproveitar; o Flávio analisa com calma.
- Ao criar nova versão: copie os arquivos atuais de `Apresentação-Completa-do-Site/` (ou outros relevantes) para `_versoes/AAAA-MM-DD/` e, se quiser, descreva no README da pasta.
